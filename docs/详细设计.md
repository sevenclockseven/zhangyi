# 账易 - 代理记账系统 详细设计

**项目代号**：zhangyi  
**版本**：v0.1（设计稿）  
**日期**：2026-06-25  

---

## 1. 项目概述

### 1.1 定位
面向个人代账会计的轻量级代理记账系统。一个人管理几十上百家客户的账务，从建账到出报表全流程覆盖。

### 1.2 核心特性
- **单机运行**：双击即用，零配置，数据本地存储
- **多账套管理**：每个代理客户独立账套，互不干扰
- **行业适配**：内置多行业会计科目模板，建账时选择行业自动加载
- **科目可更新**：会计准则变更时，推送更新包覆盖科目模板
- **报表灵活**：标准三大报表 + 自定义报表引擎
- **可扩展**：架构预留多人协作能力（SQLite → PostgreSQL 切换）

### 1.3 目标用户
- 个人代账会计（主要）
- 小型代账工作室（2-5人，未来扩展）

- 试算平衡检查：借方合计 = 贷方合计

### 3.2 科目体系

#### 3.2.1 科目结构

```
科目编码规则：
  一级：4位（1001, 1002, 2001...）
  二级：4位.2位（1002.01, 1002.02...）
  三级：4位.2位.2位（1002.01.01...）
  最多支持4级

科目分类：
  1xxx - 资产类
  2xxx - 负债类
  3xxx - 所有者权益类
  4xxx - 成本类
  5xxx - 损益类
```

| direction | enum | 借/贷 |

**模板加载逻辑：**
1. 基础模板（小企业会计准则）必选

- 已有账套可选择"同步更新"（仅追加新科目，不修改已有科目）

#### 3.3.2 凭证操作流程

```
新增凭证 → 录入分录 → 保存（草稿）
   ↓
审核（可批量） → 已审核
   ↓
记账（可批量） → 已记账（更新科目余额）
   ↓
月末结转 → 自动生成结转凭证 → 审核 → 记账
   ↓
期末结账 → 锁定本期（不允许再新增/修改）

反操作：
已记账 → 反记账 → 已审核
已审核 → 反审核 → 草稿
已作废 → 恢复 → 草稿
```

#### 3.3.3 凭证校验规则

- 借方合计必须等于贷方合计
- 必须有至少两条分录（一借一贷或多借多贷）
- 科目必须是末级科目
- 有辅助核算的科目必须填写辅助核算项
- 已记账凭证不能直接修改/删除
- 凭证日期不能在已结账期间

#### 3.3.4 凭证模板

- 支持将常用凭证保存为模板
- 模板保存分录结构（科目、摘要、辅助核算），金额留空
- 新建凭证时可从模板加载，填入金额即可
- 支持模板分类管理

### 3.4 账簿查询

#### 3.4.1 总账

- 按科目展示：期初余额、本期借方发生额、本期贷方发生额、期末余额
- 支持期间范围选择
- 支持科目级别筛选（只看一级/二级/全部）
- 点击科目可穿透到明细账

#### 3.4.2 明细账

- 单科目明细：按日期展示每笔凭证的借贷发生额和余额
- 支持辅助核算筛选（如：应收账款-按客户筛选）
- 支持期间范围选择
- 支持联查凭证（点击凭证号跳转到凭证详情）

#### 3.4.3 日记账

- 现金日记账：按日期展示现金收支
- 银行存款日记账：按银行账号分别展示
- 支持按期间导出

#### 3.4.4 多栏账

- 针对管理费用、销售费用等科目
- 按二级科目展开为多栏
- 每栏显示本期发生额

### 3.5 报表系统

#### 3.5.1 标准报表

**资产负债表：**
- 数据来源：科目余额表
- 按资产/负债/所有者权益分类
- 期末数 + 年初数对比
- 自动校验：资产合计 = 负债 + 所有者权益
- 支持导出 Excel/PDF/打印

**利润表：**
- 数据来源：损益类科目发生额
- 按月/季/年生成
- 本期数 + 本年累计数
- 支持多期间对比

**现金流量表：**
- 数据来源：凭证现金流量标记
- 主表：经营活动/投资活动/筹资活动
- 附表：间接法调整
- 需要在凭证中标记现金流量项目

#### 3.5.2 自定义报表引擎

**取数公式语法：**

```
QM(科目编码, 方向)                    -- 期末余额
QM(科目编码, 方向, 辅助核算值)         -- 期末余额（按辅助核算筛选）
QC(科目编码, 方向)                    -- 期初余额
JE(科目编码, 方向)                    -- 本期发生额
JL(科目编码, 方向)                    -- 本年累计发生额

示例：
QM("1002", "借")                     -- 银行存款期末借方余额
QM("1002", "借", "工行基本户")        -- 工行基本户期末余额
JE("6001", "贷")                     -- 主营业务收入本期贷方发生额
QM("1002", "借") - QM("1012", "借")  -- 银行存款-其他货币资金

金额格式化：
FORMAT(QM("1002", "借"), "#,##0.00")  -- 千分位格式
```

**报表模板结构：**

```json
{
  "id": "custom_01",
  "name": "费用汇总表",
  "period": "monthly",
  "rows": [

      "level": 1,
      "bold": true
    },
    {
      "label": "  工资薪金",
      "level": 2,
      "formula": "JE('6602.01', '借')"
    },
    {
      "label": "  办公费",
      "level": 2,
      "formula": "JE('6602.02', '借')"
    },
    {
      "label": "  差旅费",
      "level": 2,
      "formula": "JE('6602.03', '借')"
    },
    {
      "label": "管理费用合计",
      "level": 1,
      "bold": true,
      "formula": "JE('6602', '借')"
    }
  ]
}
```

### 3.6 期末处理

#### 3.6.1 自动转账

- 损益结转：将所有损益类科目余额结转到"本年利润"
- 支持自定义转账规则（如：按比例分配制造费用）
- 自动生成凭证草稿，审核后记账

#### 3.6.2 期末结账

```
结账流程：
1. 检查所有凭证是否已记账（有未记账凭证则提醒）
2. 执行自动转账（生成结转凭证）
3. 审核并记账结转凭证
4. 生成期末余额（作为下期期初）
5. 锁定本期（状态改为"已结账"）
6. 开启下期

反结账：
  解锁本期 → 删除结转凭证 → 可修改本期数据
```

## 4. 数据库设计

### 4.1 核心表结构

```sql
-- 账套表
CREATE TABLE account_books (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    code VARCHAR(20) UNIQUE NOT NULL,
    name VARCHAR(100) NOT NULL,
    industry VARCHAR(200),           -- 行业类型，逗号分隔（支持多行业）
    taxpayer_type VARCHAR(20),       -- small_scale / general
    start_date VARCHAR(7) NOT NULL,  -- 启用期间 YYYY-MM
    currency VARCHAR(10) DEFAULT 'CNY',
    status VARCHAR(20) DEFAULT 'active',  -- active / inactive / archived
    contact VARCHAR(50),
    phone VARCHAR(20),
    address VARCHAR(200),
    memo TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 科目表
CREATE TABLE accounts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    code VARCHAR(20) NOT NULL,
    name VARCHAR(100) NOT NULL,
    parent_code VARCHAR(20),
    direction VARCHAR(4) NOT NULL,   -- debit / credit
    level INTEGER NOT NULL DEFAULT 1,
    is_leaf BOOLEAN DEFAULT 1,
    is_system BOOLEAN DEFAULT 0,
    is_active BOOLEAN DEFAULT 1,
    aux_types VARCHAR(200),          -- 辅助核算类型，逗号分隔
    memo TEXT,
    sort_order INTEGER DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (book_id) REFERENCES account_books(id),
    UNIQUE(book_id, code)
);

-- 凭证表
CREATE TABLE vouchers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    date DATE NOT NULL,
    number VARCHAR(20) NOT NULL,
    voucher_type VARCHAR(20) DEFAULT 'general',  -- general / receipt / payment
    status VARCHAR(20) DEFAULT 'draft',          -- draft / reviewed / posted / voided
    total_debit DECIMAL(14,2) DEFAULT 0,
    total_credit DECIMAL(14,2) DEFAULT 0,
    attachments INTEGER DEFAULT 0,
    prepared_by VARCHAR(50),
    reviewed_by VARCHAR(50),
    posted_by VARCHAR(50),
    memo TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (book_id) REFERENCES account_books(id)
);

-- 凭证明细表
CREATE TABLE voucher_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    voucher_id INTEGER NOT NULL,
    line_no INTEGER NOT NULL,
    account_id INTEGER NOT NULL,
    account_code VARCHAR(20) NOT NULL,
    account_name VARCHAR(100) NOT NULL,
    debit DECIMAL(14,2) DEFAULT 0,
    credit DECIMAL(14,2) DEFAULT 0,
    memo VARCHAR(200),
    -- 辅助核算字段
    aux_customer_id INTEGER,
    aux_supplier_id INTEGER,
    aux_department_id INTEGER,
    aux_project_id INTEGER,
    aux_employee_id INTEGER,
    aux_warehouse_id INTEGER,
    aux_bank_account_id INTEGER,
    cash_flow_id INTEGER,
    FOREIGN KEY (voucher_id) REFERENCES vouchers(id) ON DELETE CASCADE,
    FOREIGN KEY (account_id) REFERENCES accounts(id)
);

-- 期初余额表
CREATE TABLE opening_balances (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    account_id INTEGER NOT NULL,
    period VARCHAR(7) NOT NULL,      -- YYYY-MM
    debit DECIMAL(14,2) DEFAULT 0,
    credit DECIMAL(14,2) DEFAULT 0,
    -- 辅助核算键（用于按维度存储期初）
    aux_key VARCHAR(200),            -- 格式：type:id，如 customer:5
    FOREIGN KEY (book_id) REFERENCES account_books(id),
    FOREIGN KEY (account_id) REFERENCES accounts(id),
    UNIQUE(book_id, account_id, period, aux_key)
);

-- 科目余额表（由凭证记账时自动更新）
CREATE TABLE account_balances (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    account_id INTEGER NOT NULL,
    period VARCHAR(7) NOT NULL,      -- YYYY-MM
    opening_debit DECIMAL(14,2) DEFAULT 0,   -- 期初借方
    opening_credit DECIMAL(14,2) DEFAULT 0,  -- 期初贷方
    period_debit DECIMAL(14,2) DEFAULT 0,    -- 本期借方发生额
    period_credit DECIMAL(14,2) DEFAULT 0,   -- 本期贷方发生额
    ytd_debit DECIMAL(14,2) DEFAULT 0,       -- 本年累计借方
    ytd_credit DECIMAL(14,2) DEFAULT 0,      -- 本年累计贷方
    closing_debit DECIMAL(14,2) DEFAULT 0,   -- 期末借方
    closing_credit DECIMAL(14,2) DEFAULT 0,  -- 期末贷方
    aux_key VARCHAR(200),
    FOREIGN KEY (book_id) REFERENCES account_books(id),
    FOREIGN KEY (account_id) REFERENCES accounts(id),
    UNIQUE(book_id, account_id, period, aux_key)
);

-- 辅助核算维度表（通用）
CREATE TABLE aux_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    type VARCHAR(20) NOT NULL,       -- customer / supplier / department / project / employee / warehouse / bank_account
    code VARCHAR(20) NOT NULL,
    name VARCHAR(100) NOT NULL,
    parent_id INTEGER,
    extra JSON,                      -- 扩展信息（联系人、电话等）
    is_active BOOLEAN DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (book_id) REFERENCES account_books(id),
    UNIQUE(book_id, type, code)
);

-- 凭证模板表
CREATE TABLE voucher_templates (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER,                 -- NULL表示全局模板
    name VARCHAR(100) NOT NULL,
    category VARCHAR(50),
    items JSON NOT NULL,             -- 模板分录结构
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 报表模板表
CREATE TABLE report_templates (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER,                 -- NULL表示全局模板
    name VARCHAR(100) NOT NULL,
    type VARCHAR(20) NOT NULL,       -- standard / custom
    config JSON NOT NULL,            -- 报表配置（行定义、公式等）
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 操作日志表
CREATE TABLE operation_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    book_id INTEGER NOT NULL,
    module VARCHAR(20) NOT NULL,     -- voucher / account / book
    action VARCHAR(20) NOT NULL,     -- create / update / delete / review / post
    target_id INTEGER,
    detail TEXT,
    operator VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
